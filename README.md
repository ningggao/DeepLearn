## Requirements
- `pip install -r requirements.txt`
- tested on Python 3.7, pytorch 1.7.1, CUDA 11.2

## Usage

- #### Run WebApp
  `python3 server.py`   
  ->then access to `localhost:5000`
  
- #### Training Model
  cpu training: `python3 train.py`

  gpu training: `python3 train.py --use_gpu`
